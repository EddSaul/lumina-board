## Quick Start

```bash
git clone https://github.com/EddSaul/lumina-board.git
cd lumina-board
npm install
cp .env.example .env
```

Add your Supabase credentials to `.env`:

```env
VITE_SUPABASE_URL=your-supabase-project-url
VITE_SUPABASE_ANON_KEY=your-supabase-anon-key
```

```bash
npm run dev
```
